---

## 📖 Overview
This repository contains my **Google Summer of Code 2026 proposal** and technical design for enhancing Apache SkyWalking BanyanDB with a robust export/import utility.

The goal is to enable efficient, scalable, and flexible data transfer across multiple formats for observability data.

---

## 🚀 Key Features
- Multi-format data export (CSV, JSON, Parquet, Binary)
- Streaming and batch data processing
- Schema mapping and evolution support
- High-performance parallel processing (worker pools)
- CLI-based export/import commands
